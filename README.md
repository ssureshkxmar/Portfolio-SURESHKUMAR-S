# ES Innovations — Portfolio

## About
Building intelligent digital systems that bridge AI, healthcare, and next-generation technology. This portfolio showcases projects in artificial intelligence, IoT, embedded systems, and advanced web applications.

## Technologies Used
- **Frontend**: React JS, Styled Components
- **Animations**: GSAP, Framer Motion, Locomotive Scroll
- **Tools**: Vite, Bun

## Project Structure
- `src/sections/Home`: Cinematic video background and landing content.
- `src/sections/About`: Professional biography and mission statement.
- `src/sections/Projects`: Interactive showcase of recent technological developments.
- `src/sections/Footer`: Navigation and professional connectivity.

## Deployment
This project is optimized for performance and premium aesthetics, designed to solve real-world challenges with futuristic engineering.

## Run Locally
```bash
bun install
bun run dev      # start dev server at http://localhost:3000
bun run build    # production build
```
